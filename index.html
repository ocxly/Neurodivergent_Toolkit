About this project

OCXLY Neuro Lab combines accessibility-focused design, lived experience, and AI-assisted development to build practical tools for neurodivergent people. Every tool is a single, self-contained page you can open, save, or host anywhere.

Educational resource only. Not medical advice and not a replacement for professional care.
