# Vision & Design Notes

Background notes on *why* OCXLY Neuro Lab exists and how its tools are built.
For the current tool list and how to run them, see the [README](README.md).

## Mission

Build tools that adapt to people, not people who must adapt to tools.

## Design principles

- **Lexend** typography (with a system-font fallback)
- High contrast, low visual clutter
- Keyboard accessible, with visible focus states
- Respects reduced-motion preferences
- Mobile friendly
- Self-contained: each tool is one page, no build step, no tracking

## Ideas being explored

- Visual Schedule Builder
- Reading Focus Mode
- Pink Noise Generator
- Energy Budget Planner
- A shared settings system across tools

Have an idea? See [CONTRIBUTING.md](CONTRIBUTING.md) — small suggestions welcome.
