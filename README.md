# ADHD Focus Helper

🚀 **Quick Start**
1. Fork or clone this repo.
2. `npm install`
3. `npm start`
- That’s it! The MVP runs locally. Low pressure, no fuss.

---

✨ **Why This Exists**

This tool is made for people who struggle with executive dysfunction, decision paralysis, and task overwhelm.  
It’s designed to be low friction so you can capture what matters quickly—no guilt, just helpful nudges.
